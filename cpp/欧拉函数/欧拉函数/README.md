# 873. 欧拉函数

给定n个正整数*a**i*，请你求出每个数的欧拉函数。

##### 欧拉函数的定义

> 1 ~ N 中与 N 互质的数的个数被称为欧拉函数，记为ϕ(*N*)。
> 若在算数基本定理中，*N*=*p**a*11*p**a*22…*p**a**m**m*，则：
> ϕ(*N*)= *N*∗*p*1−1*p*1∗*p*2−1*p*2∗…∗*p**m*−1*p**m*

#### 输入格式

第一行包含整数n。

接下来n行，每行包含一个正整数*a**i*。

#### 输出格式

输出共n行，每行输出一个正整数*a**i*的欧拉函数。

#### 数据范围

1 ≤ *n* ≤ 100,
1 ≤ *a**i* ≤ 2∗10^9

#### 输入样例：

```
3
3
6
8
```

#### 输出样例：

```
2
2
4
```
