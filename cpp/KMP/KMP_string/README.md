# 831. KMP字符串

给定一个模式串S，以及一个模板串P，所有字符串中只包含大小写英文字母以及阿拉伯数字。

模板串P在模式串S中多次作为子串出现。

求出模板串P在模式串S中所有出现的位置的起始下标。

#### 输入格式

第一行输入整数N，表示字符串P的长度。

第二行输入字符串P。

第三行输入整数M，表示字符串S的长度。

第四行输入字符串M。

#### 输出格式

共一行，输出所有出现位置的起始下标（下标从0开始计数），整数之间用空格隔开。

#### 数据范围

1 ≤ *N* ≤ 10^4

1 ≤ *M* ≤ 10^5

#### 输入样例：

```
3
aba
5
ababa
```

#### 输出样例：

```
0 2
```
