# 883. 高斯消元解线性方程组

输入一个包含n个方程n个未知数的线性方程组。

方程组中的系数为实数。

求解这个方程组。

下图为一个包含m个方程n个未知数的线性方程组示例：

![9a504fc2d5628535be9dcb5f90ef76c6a7ef634a.gif](https://cdn.acwing.com/media/article/image/2019/06/27/19_b30080c698-9a504fc2d5628535be9dcb5f90ef76c6a7ef634a.gif)

#### 输入格式

第一行包含整数n。

接下来n行，每行包含n+1个实数，表示一个方程的n个系数以及等号右侧的常数。

#### 输出格式

如果给定线性方程组存在唯一解，则输出共n行，其中第i行输出第i个未知数的解，结果保留两位小数。

如果给定线性方程组存在无数解，则输出“Infinite group solutions”。

如果给定线性方程组无解，则输出“No solution”。

#### 数据范围

1 ≤ *n* ≤ 100,
所有输入系数以及常数均保留两位小数，绝对值均不超过100。

#### 输入样例：

```
3
1.00 2.00 -1.00 -6.00
2.00 1.00 -3.00 -9.00
-1.00 -1.00 2.00 7.00
```

#### 输出样例：

```
1.00
-2.00
3.00
```
